# GachiC

Gachimuchi-themed programming language (Work in progress)

## Building from source

### Dependencies

##### POSIX

* cmake >= 3.13.4
* gcc >= 9.3.0 or clang >= 10.0
* LLVM, Clang, LLDF libs == 10.

##### Windows

* cmake >= 3.13.4
 * Microsoft Visual Studio. Supported versions:
   - 2017 (version 15.8)
   - 2019 (version 16)
* LLVM, Clang, LLDF libs == 10.

#### Instructions

##### POSIX

```sh
$ mkdir build
$ cd build
$ cmake ..
$ make
```
